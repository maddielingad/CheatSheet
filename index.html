Common HTML Tags

Tag Name Description
<p> Paragraph Represents a paragraph.
<body> Body Contains all the contents of an HTML document.
<div> Division Used as a container that divides an HTML document into sections.
<span> Span Used as an inline container for text and can be used to group text for styling purposes.
<h1>-<h6> Heading Represents the six levels of headings in HTML from most important to least important.
<strong> Strong Displays the content inside in bold.
<em> Emphasis Italicizes the content inside.
<br> Line Break Creates a line break in text. Requires only an opening tag.
<li> List Creates a list of items inside ordered lists and unordered lists.
<ol> Ordered List Creates a list of items in a numbered, sequential order.
<ul> Unordered List Creates a list of items that do not have a numerical ordering.

Multi-Media Tags

Tag Attributes Name Description
<a> href='url' Anchor Defines hyperlinks.
<img> controls src='url' Image Embeds images in an HTML document.
<video> controls src='url' Video Embeds a media player for video playback.
<audio> controls src='url' Audio Used to play an audio file.

Semantic Tags

Tag Name Description
<header> Header Represents a container for introductory content.
<nav> Navigation Defines a set of navigational links.
<main> Main Specifies the main content of a document.
<footer> Footer Defines a footer for a document or section.
<section> Section Defines a section in a document.
<article> Article Represents content that makes sense on its own.
<aside> Aside Represents secondary content.

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.
